C#과 .NET을 활용한 서버-클라이언트 통신 실습 프로젝트입니다.

효율적인 패킷 관리와 확장성 있는 서버 구조를 설계하는 데 중점을 두었습니다.

이 프로젝트는 Google Protobuf를 이용해 패킷을 정의하고, 자동화를 통해 서버의 통신 로직을 구조화하는 과정을 담고 있습니다.


Server: 객체 지향적인 설계와 자동화 코드 생성기를 도입하여 구조적 완성도를 높였습니다.

Client: 서버의 기능을 테스트하고 데이터를 수신하기 위한 간단한 형태로 구현되었습니다.

PacketGenerator에서 빌드하면 PacketManager가 나오고 이걸 서버에 적용하는 방식입니다.
